# 创业探讨

> 独立于工作助手的创业讨论空间。
> 此文件夹与 `.claude/` 体系完全分离，不触发10步行为引擎。

---

## 触发机制

以下方式会自动识别并进入创业讨论模式：

**触发词：**
- 消息包含 "创业"、"商业模式"、"副业"、"自己做"、"机会分析" 等关键词
- 消息以 `💡`、`🚀` 开头

**手动触发：**
- 直接提及 "创业探讨"、"创业讨论"

自动识别后，主助手的10步工作流程被跳过，直接进入此上下文讨论。

---

## 目录结构

| 目录 | 用途 |
|:-----|:-----|
| `Context.md` | 当前讨论上下文、已有想法进度 |
| `Ideas/` | 具体创业想法记录 |
| `Analysis/` | 市场分析、可行性、竞品分析 |
| `Research/` | 调研笔记、资料收集 |

## 讨论原则

1. **基于已知事实**：对公司、行业、个人的分析基于已记录的信息
2. **不推测不编造**：缺少的信息标注【需确认】，不确定的标注【推测】
3. **务实导向**：聚焦可行性和落地路径，不做空泛讨论
4. **独立存储**：所有讨论记录仅存于此文件夹，不混入工作助手体系
