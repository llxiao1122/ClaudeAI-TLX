# 安全责任链

> 安全管理体系和责任主体映射。
> 来源：`02-安全管理/` 系列制度

---

## 责任层级

```
总经理(第一责任人)
  └── 分管副总(分管范围责任)
        └── 物资部长(部门责任)
              └── 副主任(安全) ← 安全员(日常监督)
                    └── 工班长(班组现场责任)
                          └── 库区负责人(分区责任)
                                └── 保管员(岗位责任)
```

## 安全工作主线

```
安全方针目标 → 安全生产责任制 → 安全教育培训 → 安全检查 → 隐患排查治理 → 应急演练 → 考核奖惩
```
