# 供应链核心流程

> 物资从计划到处置的全链路流程与控制点。
> 来源：`01-物资管理制度/` 系列文件、`03-工作指导手册/` 系列

---

## 总流程

```
计划编制 → 采购审批(电商/招标) → 物资编码 → 到货验收 → 入库上架 → 仓储管理(6S/巡查) → 出库领用 → 账务核对 → 回收/报废处置
   ↑                                                                                                              |
   └──────────────────────── 考核评估 ← 修复/周转件管理 ← 捐赠管理 ← 旧版归档 ←─────────────────────────────────────────┘
```

## 各环节控制点

| 环节 | 控制要求 | 依据文件 |
|------|----------|----------|
| **编码** | 必须先编码后采购 | `01-01/物资编码管理细则-A2` |
| **采购** | 电商采购 vs 招标采购分路径 | `01-01/电子商务采购管理规定-A3` |
| **验收** | 入库前按标准验收 | `01-02/物资验收入库管理规定-A1` + `03-02/物资验收及入库作业指导手册` |
| **入库** | 验收合格上架，WMS记录位置 | `01-02/物资仓储管理办法-B1` + `03-02/WMS入库操作手册` |
| **仓储** | 6S标准+巡查+温湿度监控 | `07-现场管理与6S/` 系列 |
| **出库** | 领用审批→系统出库→实物出库 | `03-02/WMS出库流程操作手册` |
| **账务** | 定期盘点核对，差异分析 | `09-评估表/系统账务核对及差异分析评分表` |
| **回收处置** | 危废与报废独立处置路径 | `03-01-危废与报废/` 系列 |
| **考核** | 季度量化考核 | `01-03/物资管理考核细则-B1` |

## 例外路径

| 场景 | 走什么流程 | 依据 |
|------|-----------|------|
| 电商采购 | 电子商务采购规定（非招标路径） | `电子商务采购管理规定-A3` |
| 捐赠物资 | 捐赠物资管理细则（专项备案） | `捐赠物资管理细则-A1` |
| 修复件/周转件 | 修复件和周转件管理细则（不入主流程） | `修复件和周转件管理细则-A0` |
| 建设移交备件 | 建设移交备品备件接管指导手册（批量接管） | `03-03/建设移交备品备件接管指导手册` |
