# 审批逻辑

> 企业审批流程的设计原则和运行逻辑，平衡效率与风控。

## 一、审批原则

- **权责对等**：审批权与责任相匹配
- **分级授权**：按金额/重要性分层审批
- **程序合规**：符合制度和法规要求
- **效率优先**：减少不必要的审批层级

## 二、审批权限设计

| 层级 | 审批范围 | 审批人 |
|------|---------|-------|
| 部门级 | 日常事务、小额支出 | 部门负责人 |
| 分管级 | 专项工作、中等额度 | 分管领导 |
| 班子级 | 重大事项、大额支出 | 班子会/党委会 |
| 上级级 | 特别重大事项 | 上级单位 |

## 三、审批要素

- **事实依据**：事项背景、必要性
- **方案论证**：可行性、备选方案
- **风险评估**：潜在风险及应对
- **合规审查**：是否符合制度和法规
- **预算保障**：资金来源和额度

## 四、常见问题

- 审批流程过长影响效率
- 审批流于形式、走过场
- 权责不对等（有审批权无责任感）
- 紧急事项的绿色通道缺失

```
工班提报 → 主任(王敬宇) → 部长(李彦奎) → 分管副总(商丽娜) → 总经理(郝晓平)
```

### 物资报废

```
保管员申请 → 主任 → 部长 → 资产组(荆幸斌) → 财务
```

### 安全奖惩

```
安全员提报 → 副主任/安全(卢丽英) → 部长(李彦奎) → 分管副总(商丽娜)
```

### 公文制发

```
起草 → 科室核稿 → 会签 → 领导签发 → 编号 → 印发
```
