## 七、AI引用策略

### 优先级规则

```
直接引用：A2 > A1 > A0（同系列数字越大越新）
          B1 > A5（跨系列B>A）
          2026版 > 2025版 > 2024版 > ...（含年度文件）
科室发文：编号越大越新（如40号修订19号）
手册/制度关系：制度定义规则优先，手册定义操作参考
```

### 具体策略

1. **有明确版本号且无冲突** → 直接引用最高版本
2. **有版本号但旧版未归档** → 回答中注明"以X版为准"
3. **无版本号** → 标记 **需人工确认**，AI不得自行推断
4. **同名双文件** → 标记 **需人工确认**，AI不得自行选择
5. **制度与手册都涉及同一主题** → 制度定原则+手册定操作，互补引用
6. **科室发文有修订关系** → 引用最新编号（如2023-40号取代2022-19号）
7. **试行版文件** → 需注明"试行"，提示可能被正式版替代

### 快速判断口诀

```
看年份 → 看版本号(A0/B1/A3) → 看编号(科室发文) → 查冲突表
年份最新 + 版本最高 = 有效版
无年份无版本 = 需人工确认
同名两文件 = 需人工确认
```
