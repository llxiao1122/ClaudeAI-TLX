# 台账每日提醒命令

## 用途
生成当天台账核查提醒并更新 `context/today.md`

## 用法
在对话中输入：
```
!python3 .claude/Commands/台账提醒/gen_daily_check.py
```
然后将输出内容粘贴到 `today.md` 的「今日台账核查」章节。

## 自动流程（由 AI 执行）

当你收到用户消息后，在执行 10 步流水线时：
1. 运行 `gen_daily_check.py` 生成当天提醒
2. 将提醒内容更新到 `context/today.md` 的「今日台账核查」章节
3. 如果当天有特定日期检查（如18日、周五等），在「今日紧急」中补充提醒
