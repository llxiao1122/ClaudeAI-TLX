# Workflow：工作安排生成

> 生成每日工作安排的标准流程

---

## 输入

- 日期（某月某日）
- 特殊事项（用户告知的临时工作）
- 领导安排（如有）

## 流程

### Step 1：读取Context
```
读 context/today.md
  → 获取：日期/星期/值班人/距周五天数
```

### Step 2：查询值班表
```
查 知识库/04-工作台账与工具/2026-值班表模板.xlsx
  → 确认值班人（不推算，必须查原始数据）
```

### Step 3：读取待办清单
```
读 memory/tasks.md
  → 获取：紧急待办 / 常规待办 / 持续关注
  → 筛选：截止日<=当天+3天的优先列出
```

### Step 4：查询人员档案
```
读 memory/organization.md
  → 确认各人库区分配（7月轮换）
读 references/人员档案.md
  → 确认人员特点和适合的工作类型
```

### Step 5：安全重点
```
查 context/today.md → 今日安全重点
查 operations.md → 防汛/周期安全事项
```

### Step 6：生成安排

按以下结构输出：

```
一、基本信息：日期/星期/值班人/截止倒计时
二、台账重点：每日必填 + 本周到期 + 本月倒计时
三、分工表：责任人 | 工作 | 台账关联 | 备注
四、安全重点：今日专项安全事项
五、晚立岗确认清单：逐项打勾
```

### Step 7：引用模板
```
调 templates/工作安排模板.md
  → 填充内容
```

### Step 8：更新Context
```
写 context/today.md
  → 更新今日重点/风险预警
```

---

## 检查清单

- [ ] 值班人是否查了xlsx原始数据？（禁止推算）
- [ ] 待办清单是否查了最新版本？
- [ ] 人员分配是否与当月库区轮换一致？
- [ ] 安全重点是否覆盖防汛/消防/专项？
- [ ] 截止日倒计时是否正确？
- [ ] 是否标注了不确定项？

## 关联

- Memory: context/today.md, tasks.md, organization.md
- Knowledge: 值班表xlsx, operations.md
- Template: 工作安排模板.md
