## 二、物资出入库管理制度

### 入库流程
1. **接货准备** — 保管员根据到货预报清理货位
2. **验收核对** — 双人核对实物与单据（品名/规格/数量/质量）
3. **质量鉴定** — 需质检的物资通知相关部门检验
4. **系统录入** — 合格物资录入CIMS系统，生成入库单
5. **上架摆放** — 按库区规划摆放，更新货位卡
6. **单据归档** — 送货单/质检单/入库单整理归档

### 出库流程
1. **领料审核** — 核对领料单审批签字、印章齐全
2. **拣货出库** — 保管员按单拣货，双人核对
3. **系统出库** — 录入CIMS系统，生成出库单
4. **物资交接** — 领料人签字确认
5. **单据归档** — 出库单/领料单整理归档

### 退库流程
1. **退库申请** — 使用部门填写退库单
2. **质量鉴定** — 检查物资状态，确定可利用/报废
3. **系统录入** — 退库入库，更新库存
4. **归位上架** — 可用物资归位，报废物资隔离存放
5. **单据归档** — 退库单/鉴定记录归档

### 调拨流程
1. **调拨指令** — 接收调拨通知
2. **拣货出库** — 按调拨单拣货
3. **系统调拨** — 录入CIMS系统
4. **运输跟踪** — 跟踪调拨物资到达情况
5. **到货确认** — 接收方确认后闭环

| C区 | 办公用品/劳保用品 | [待分配] |
| 杂品库 | 杂品/消耗品 | [待分配] |
| 应急物资区 | 防汛/消防/应急物资 | [待分配] |

### 物资摆放规范
- **五五摆放** — 按五五成行/成方/成串/成包原则摆放
- **四号定位** — 库号/架号/层号/位号统一标识
- **标识要求** — 物资卡清晰完整，信息包括：名称/规格/数量/入库日期/有效期
- **通道要求** — 主通道≥1.5m，消防通道禁止堆放

### 库区安全要求
- **防火：** 灭火器在位有效、消防通道畅通、严禁烟火
- **防潮：** 温湿度记录、物资离地≥15cm、通风设备运行正常
- **防盗：** 门禁系统正常、监控覆盖、下班落锁检查
- **防鼠防虫：** 定期检查、投放药剂记录

## 四、人员管理制度

### 排班轮岗制
- **主责区管理** — 每人固定负责一个区域的基本管理
- **轮岗周期** — 每季度轮换一次区域，确保全员熟悉全库
- **值班安排** — 每日安排1名值班保管员负责当日巡查和应急响应

### 绩效考核维度
| 维度 | 权重 | 考核标准 |
|-----|------|---------|
| 工作质量 | 40% | 出入库准确率、台账规范率、盘点差异率 |
| 工作效率 | 20% | 任务按时完成率、响应速度 |
| 工作纪律 | 20% | 出勤、着装、制度遵守 |
| 团队协作 | 10% | 配合度、互助意识 |
| 安全规范 | 10% | 安全操作、隐患排查 |

### 班组会议制度
- **每日晨会**（10分钟）— 当日任务分配+安全提醒
- **每周例会**（30分钟）— 台账汇总+问题通报+下周计划
- **月度总结**（60分钟）— 绩效考核+盘点分析+改进计划

## 五、应急预案

### 物资紧急出库
1. 接到紧急领料通知
2. 工班长立即指派专人
3. 优先出库，后补手续
4. 事后24小时内补齐单据

### 库区突发事故
1. **发现人**立即报告工班长
2. **工班长**判断情况 → 上报物资调配室
3. 启动相应应急预案（火灾/水灾/设备事故等）
4. 保护现场，留取证据
5. 填写事故报告，分析原因，制定防范措施

### 系统故障（CIMS无法使用）
1. 启用纸质台账记录
2. 工班长登记故障时间、受影响业务
3. 联系信息中心处理
4. 系统恢复后补录电子数据
