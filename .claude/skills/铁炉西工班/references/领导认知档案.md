> 创建时间：2026-07-04

---

## 使用规则

1. **不编造** — 无证据的行为/习惯/偏好标记为"未知"
2. **只做可观测推断** — 基于：指令文本 / 历史任务记录 / 沟通反馈
3. **三层区分** — 每条记录标注类型：
   - `【事实】` = 明确指令/行为/语言
   - `【推测】` = 基于多次事实的合理归纳
   - `【不确定】` = 出现不足3次的现象
4. **稳定偏好确认标准** — 同一行为模式出现≥3次方可记为偏好
5. **冲突处理** — 指令不一致时：优先最近指令 / 优先直接上级 / 标注冲突并确认

## 附录：沟通状态标签参考库

以下为可用的标准标签，仅当可观测行为匹配时才可使用：

| 标签 | 典型可观测表现 |
|------|--------------|
| **紧急强化型** | 强调时间节点、反复催办、使用"立即""马上"等措辞 |
| **稳定推进型** | 给出明确时限但不催促、接受分阶段反馈、关注整体进度 |
| **高关注检查型** | 要求过程汇报、中途查看进展、关注中间结果 |
| **结果导向型** | 只问最终结果、不问过程细节、关注完成与否 |
| **细节追问型** | 反复询问具体数据/操作细节/流程步骤 |

> 禁止使用心理描述类标签（如"焦虑型""急躁型""不满型"），仅使用行为描述标签。
