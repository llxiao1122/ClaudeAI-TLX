# 工班长每日台账核查提醒

**背景：** 铁炉西工班有 20 项安全台账需要按不同周期核查（每日/每周五/每月固定日/季度末）。
源文件：`Knowledge/04-工作台账与工具/工班长每日核查台账清单.xls`

**提醒规则：**

| 周期 | 台账项 | 触发日 |
|:----|:-------|:-------|
| 每日 | [01]消防安全巡查、[16]断电登记 | 每天 |
| ~~已移交~~ | ~~[06]叉车日管控、[15]叉车使用记录~~ | ~~已移交车辆中心~~ |
| 每周五 | [09]防寒防汛、[11]巡更、[18]安全自查、[19]隐患排查、[20]CIMS故障 | 周五 |
| 每月15日/30日 | [02]防火检查记录表 | 15日、30日 |
| 每月18日 | [03]灭火器统计、[08]杂品库物资、[09]防寒防汛 | 18日 |
| 每月20日 | [12]应急演练台账 | 20日 |
| 每月25日 | [10]安全培训、[14]充电记录表 | 25日 |
| 季度末 | [05]防爆手电筒充电记录 | Q末最后一个工作日 |

**如何应用：**
- 每天更新 `today.md` 时，运行 `python3 .claude/Commands/台账提醒/gen_daily_check.py` 生成当日提醒
- 将输出内容嵌入 `today.md` 的「今日台账核查」章节
- 遇到节假日（核查日在周末），自动提示提前至前一个工作日
- 16:00 前跟进台账进度（靳攀/谭继衡）

**相关文件：**
- [[daily-context]] — today.md 中已有「今日台账核查」章节
- 台账清单参考：`Knowledge/04-工作台账与工具/工班长每日核查台账清单-提醒参考.md`

**Why:** 20项台账有不同核查周期，靠人工记忆容易遗漏。系统化提醒确保每天不漏项。
**How to apply:** 每次处理 `today.md` 时自动运行生成脚本，把当日需查项更新到 context。
