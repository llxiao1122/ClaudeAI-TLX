### 包保制度

铁炉西工班由 **董文静（管理岗）** 进行包保。包保人负责对所包保工班的日常工作进行监督、指导和考核，确保制度执行到位、问题及时整改。

### 日常工作对接

- **业务汇报、任务接收、绩效考核** → 王敬宇（直接上级）
- **包保监督、制度执行、问题整改** → 董文静（包保人）
- **安全检查、隐患排查、安全培训** → 卢丽英（安全对口）/ 王亮 / 刘欢
- **危废/废旧物资管理** → 荆幸斌 / 王超
- **工班内勤业务（考勤/公文/简报）** → 张梦圆（科室内勤岗）→ 陈红洁对接
- **制度修订、管理改进、专项报告** → 张新宇 / 李彦奎（上行层级）
- **重大事项、跨部门协调** → 商丽娜（分管领导审批层级）
- **最高决策、战略方向** → 郝晓平（总经理）

- 物资仓储管理办法（B1版）签署页：起草人含王敬宇、李彦奎；审核人商丽娜；签发人郝晓平

## 二、物资管理部岗位职责说明

> 来源：物资管理部岗位说明书（2026年6月版）

### 物资调配室岗位体系

```
主任（管理序列）→ 全面管理
 └── 副主任（管理序列）→ 协助管理
      ├── 库存管理岗（职能序列）→ 标准化/盘点/培训/危废处置
      ├── 仓库保管工班长（技能序列）→ 工班全面管理
      │    └── 仓库保管员（技能序列）→ 物资收发存/巡查/6S
      ├── 安全管理岗（职能序列）→ 科室安全管理/培训/应急
      └── 资产管理员（职能序列）→ 报废鉴定/废旧处置
```

### 各岗位职责与任职条件

#### 仓库保管工班长（李林骁的正式岗位）

**职责：**
1. 组织物资验收入库、出/移库、调拨、盘点及库存台账管理
2. 组织所属线路废旧物资接收保存及配合处置
3. 五型班组管理
4. 组织业务培训、安全规章制度学习
5. 工班人员绩效考核
6. 配合建立健全仓储安全/危废/作业制度

**任职条件：** 全日制专科及以上 / 5年经验3年相关 / 熟悉仓储物资管理 / 熟练操作特种设备和计算机

#### 仓库保管员（杨梦卓/陈红洁/张志斌/苗笑天/谭继衡的正式岗位）

**职责：**
1. 到货物资验收、入库、不合格品退货、日常盘点、废旧物资回收保管、设备维护保养、库区物资摆放及卫生
2. 按设备安全操作规程作业，异常及时上报
3. 借用单据归档、跟进借用物资归还
4. 6S标准化建设
5. 巡库、消防设施巡查、问题上报跟进

**任职条件：** 全日制专科及以上 / 5年经验2年相关 / 熟悉仓储物资管理 / 操作特种设备和计算机

#### 库存管理岗（对应科室管理职责）

**职责：**
1. 到货验收/入库/出库工作标准制定、设备安全操作规程制定、库存消耗分析、特种设备操作证复审、WMS系统优化
2. 新线物资总库筹备、建设移交物资接管
3. 调拨管理、盘点计划制定
4. 呆滞物资/效期物资/危废鉴定、危废处置合同、危废回收处置
5. 仓储管理类标准化文本编制修订、培训计划制定

#### 安全管理岗（对应王亮的岗位）

**职责：**
1. 建立健全部门安全管理类制度
2. 部门年度安全工作计划
3. 应急预案、年度演练计划、组织应急演练
4. 安全培训、新员工三级安全教育
5. 隐患排查、检查问题督促整改、危险作业卡控
6. 双重预防体系、风险辨识

#### 资产管理员（对应荆幸斌的岗位）

**职责：**
1. 组织拟报废资产技术鉴定、出具鉴定意见、组织实物交接并建台账
2. 调研废旧物资处置单位、建立处置单位库、合同签订、收款、组织实物交接

### 岗位序列说明

| 序列 | 岗位 | 特点 |
|:---:|------|------|
| **管理序列** | 主任、副主任 | 制度制定/体系建立/全面管理 |
| **职能序列** | 库存管理岗、安全管理岗、资产管理员 | 专业职能/标准制定/培训组织 |
| **技能序列** | 工班长、保管员 | 现场作业/执行落实/设备操作 |

> 工班长（技能序列）虽不属管理序列，但实际承担人员管理和绩效考核职责。日常工作中对上对接管理序列（主任/副主任）和职能序列（库存管理岗/安全管理岗），对下管理技能序列（保管员）。

### 对管理策略的影响

| 原建议 | 变更后 |
|-------|--------|
| 杨梦卓带教张志斌，通过师父带动徒弟 | 不再依赖师徒链，转向制度管理+明确标准 |
| 陈红洁是工班长的徒弟，保持沟通 | 保持现有自主工作模式，师徒关系不再作为管理依据 |
| 苗笑天受谭继衡早期带教影响 | 历史影响仍在但不再作为管理路径 |

### 替代管理路径

师徒关系结束后，管理转向以下方式：
- **情境领导理论** — 根据每个人的能力×意愿水平适配管理风格（D1-D4）
- **制度管理** — 台账看板、任务清单、透明化标准
- **正向引导** — 对表现好的给予认可，建立正向循环

**Why:** 用户明确告知所有师徒带教均已结束，后续管理不再以此为依据。
**How to apply:** 人员分工和日常管理中不再使用"师父带徒弟"的逻辑，改用"情境领导+制度管理"。

## 四、Skill核心定位反馈

**反馈内容：** 李林骁Skill主要用来记录人员情况以及工作任务安排，并非用来生成表格。

**Why:** 用户明确表示不需要排班表/台账生成/任务排序之类的表格输出功能，需要用记录和跟踪的方式管理班组。

**How to apply:**
- 所有交互以"记录人员档案"和"记录/跟踪工作安排"为核心输出方式，而非生成格式化表格。
- 脚本工具专注于数据记录和查询，不输出可视化表格。
- 快速响应模板以"记录"和"跟踪"为关键词，不以"生成"为关键词。
- references 作为知识库/档案库使用，scripts 作为记录和查询工具。
- 涉及 [[铁炉西工班]] 的岗位要求。
