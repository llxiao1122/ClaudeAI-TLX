## 一、三菱电梯赠送备件 — 验收与存放管理

### 工作来源

上海三菱电梯有限公司向客运中心赠送一批电梯备件，由物资管理部牵头，**铁炉西工班负责验收**。

### 会议纪要（2025年12月5日）

**地点：** 铁炉西车辆段物资总库
**参会：** 商丽娜、任艳江、李彦奎、包世臣、段沛延、王红义、**李林骁**、三菱电梯厂家代表（冉鹏飞、刘少磊）

**会议确定事项：**

| 事项 | 内容 |
|------|------|
| **移交方式** | 电缆以套为单位接收，单套附带明细清单+厂家印章；厂家于12月31日前完成归类清点包装 |
| **验收标准** | 按《捐赠物资管理细则》和《物资验收入库管理规定》执行 |
| **瑕疵处理** | 存在污渍锈蚀/非全新时，厂家清理保养，工程师确认功能不受影响后方可验收 |
| **合格证明** | 须附带产品合格证明和物资存放要求说明（温湿度/保养周期/保存期限） |
| **包装要求** | 厂家外包装或二次包装，避免裸露；成套备件整套封装 |
| **定期维保** | 客运中心做好本次赠送备件的定期维修保养方案 |
