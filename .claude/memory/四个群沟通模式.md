# 四个群沟通模式

## 物资调配室群

| 项目 | 内容 |
|:----|:------|
| **核心功能** | 每日考勤立岗汇报 + 董文静/张梦圆发通知 |
| **铁炉西汇报人** | 李林骁在=李林骁发，不在=**杨梦卓**自动代发 |
| **消息过滤** | 只看铁炉西考勤+董文静通知，其他工班考勤全跳过 |
| **日均耗时** | 30秒 |

## 危废废旧群

| 项目 | 内容 |
|:----|:------|
| **核心功能** | 危废回收/处置全流程跟踪 |
| **核心人物** | **王超**（危废系统操作+联单+催办）、**荆幸斌**（废旧处置分批+看货+踏勘） |
| **消息密度** | 🔴 极高，全科室所有工班 |
| **消息过滤** | 只看**@李林骁**和**@所有工班长**，其余跳过 |
| **处罚特点** | 王超和荆幸斌频繁用"纳入考核"施压 |
| **回收流程** | 王超分配任务→涉及工班处理→发照片汇报→系统录入联单→王超确认 |
| **日均耗时** | 1-2分钟 |

## 安全管理群

| 项目 | 内容 |
|:----|:------|
| **核心功能** | 气象预警+安全通知+培训+专项排查 |
| **核心人物** | **王亮**（占消息量80%）、刘欢（会议/演练）、卢丽英（偶尔直接安排） |
| **消息过滤** | **全看**——@所有人≈与铁炉西有关 |
| **月度固定节点** | 每月9日步行生命通道、每月30日前疏散通道排查、每月下旬安全例会 |
| **日均耗时** | 1-2分钟 |

## 台账更新群（科室大群）

| 项目 | 内容 |
|:----|:------|
| **核心功能** | 科室制度意见征询+专项工作通知+督办 |
| **核心人物** | **张梦圆**（信息中转站，最高频）、**靳攀**（专项检查）、**王红义**（专项检查） |
| **消息过滤** | 需要逐条判断：张梦圆的制度征询（无意见可不回），靳攀/王红义的专项工作需处理 |
| **日均耗时** | 1-2分钟 |
