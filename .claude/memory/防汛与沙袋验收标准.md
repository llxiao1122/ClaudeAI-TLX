# 防汛与沙袋验收标准

## 主汛期防汛巡查

- **频次：** 下雨期间各工班需汇报库区巡视情况
- **汇报格式：** "XX车辆段目前XX雨，各库区巡视无异常" + 现场照片
- **铁炉西执行人：** **苗笑天**（已替他完成两次汇报，格式规范）
- **发现漏雨时：** 铺设防水材料+放置物料盒+在汇报中注明
- **近期漏雨工班参考：** 关陈（立体库）、圃田（中小件区3处+大件区4处）、郑东（大件区）、南环（暂存区）

## 沙袋验收标准

来源：王敬宇明确指令（针对2026年7月沙袋到货）

| 条件 | 标准 |
|:----|:----|
| 正常重量范围 | **15-17kg** |
| 沙子潮湿（重量偏高） | **18kg以内可接收，接收后晾晒处理** |
| 超18kg | **退供应商换货** |
| 抽样比例 | **10%**（如到货400个，抽40个） |
| 抽样方法 | 随机抽样称重 |
| 送货单数量 vs 实际到货 | 逐项核对 |

> 以后铁炉西到货沙袋时直接参照此标准，不再需要确认。

## 气象预警应对

王亮（安全管理岗）负责转发气象预警+运营公司响应措施：

| 预警等级 | 响应要求 |
|:--------|---------|
| 暴雨黄色/橙色预警 | 各库区大门摆放防汛物资，值班人员保持电话畅通 |
| 高温橙色预警（≥37℃） | 室外作业不超过6小时/天，高温3小时停止室外作业 |
| 雷暴大风黄色预警 | 下班前摆放防汛物资，值班人员保持通信 |

## 安全管理群沟通模式

- **王亮是核心发信人**——气象预警+安全通知
- 各工班统一格式汇报汛情
- 与非铁炉西相关的汇报直接跳过
