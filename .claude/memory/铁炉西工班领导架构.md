# 铁炉西工班上级领导组织架构

## 汇报层级链

```
总经理 郝晓平
 └── 分管领导 商丽娜（分管物资管理中心 + 财务部）
      └── 物资管理中心部长 李彦奎
           └── 物资管理中心副部长 张新宇
                └── 物资调配室主任 王敬宇（主持工作，分管业务）

```

## 包保制度

铁炉西工班由 **董文静（管理岗）** 进行包保。包保人负责对所包保工班的日常工作进行监督、指导和考核，确保制度执行到位、问题及时整改。

## 日常工作对接

- **业务汇报、任务接收、绩效考核** → 王敬宇（直接上级）
- **包保监督、制度执行、问题整改** → 董文静（包保人）
- **安全检查、隐患排查、安全培训** → 卢丽英（安全对口）/ 王亮 / 刘欢
- **危废/废旧物资管理** → 荆幸斌 / 王超
- **工班内勤业务（考勤/公文/简报）** → 张梦圆（科室内勤岗）→ 陈红洁对接
- **制度修订、管理改进、专项报告** → 张新宇 / 李彦奎（上行层级）
- **重大事项、跨部门协调** → 商丽娜（分管领导审批层级）
- **最高决策、战略方向** → 郝晓平（总经理）

## 注意事项

- 涉及物资调配室的制度流程问题，优先向王敬宇汇报
- 涉及安全方面的台账、检查、整改，需同时知会卢丽英、王亮
- 涉及危废/废旧物资的处置管理，对接荆幸斌、王超
- 涉及跨部门协调或资源申请，经王敬宇→李彦奎→商丽娜逐级上报
- 领导之间有分管交叉的事项，以王敬宇为统一入口，由其协调
