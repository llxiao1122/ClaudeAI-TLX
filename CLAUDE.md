# 铁炉西工班管理

## 📋 项目定位
铁炉西工班管理辅助系统。本仓库在 MacBook（lee）和 Mac mini（mi）之间共享。

## 🚀 每次启动自动执行
1. **加载 `.claude/MEMORY.md`** — 获取记忆索引
2. **加载 `.claude/memory/` 下所有 `.md` 文件** — 恢复完整上下文
3. **检查 `待办任务清单.md`** — 关注紧急截止项

## 🔄 跨设备同步规范

### 切换设备时
```bash
git pull                          # 拉取另一台设备更新的记忆
```
### 更新记忆后
```bash
git add .claude/memory/ .claude/MEMORY.md
git commit -m "记忆更新"
git push
```

### 各设备独立（不同步）
- `.claude/settings.local.json` — 权限配置
- `.claude/scheduled_tasks.*` — 定时任务
- `./笔记/` — 本地草稿
